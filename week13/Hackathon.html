Which Industry is the Most Profitable?¶

In [188]:
industries_sum = {}
industries_mean = {}
for key, value in sheet1.groupby('Industry'):
    industries_sum[key] = value['Total BRR'].count()
    industries_mean[key] = value['Total BRR'].mean()
labels = sorted(industries_sum.keys())
ixs = 5 * np.arange(len(labels))
width = 2
bars1 = [industries_sum[k] for k in labels]
bars2 = [industries_mean[k] for k in labels]
fig, axarr = plt.subplots(2, 1)
axarr[0].bar(ixs, bars1, width)
axarr[0].set_xticks(ixs + width * 0.5)
axarr[0].set_xticklabels(labels)
axarr[1].bar(ixs, bars2, width)
axarr[1].set_xticks(ixs + width * 0.5)
axarr[1].set_xticklabels(labels)
plt.show()

These graphs tell us a couple things.

  1. One or two industries have a ton of businesses.
  2. The average amount earned for these large industries is generally lower.

In other words, we can assume that for the industries that have more companies it is "easier" to land a contract, but on average it pays less.

We can scatter this data and essentially draw a "classification line".

Now let's be fancy¶

We can define an expected value based on the average amount earned from each industry determined above.

In [191]:
industry_averages = {labels[i]:bars2[i] for i in range(len(labels))}
multipliers = {'High Risk': 0.25,
               'Low Risk': 0.75,
               'M':0.5}
ex_vals = []
for row in sheet3.values:
    try:
        ex = industry_averages[row[2]] * multipliers[row[-1]]
    except KeyError:
        ex = 0
    ex_vals.append(ex)
df = sheet3
df['EX'] = ex_vals
